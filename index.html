Git Commands

Creating Repo

git init -- this command initiate the repo with .git file in the root folder which will track all of its child itmes

Adding single Files

git add index.js

Adding mulitple Files

git add .

Removing files from stage(Single File)

git rm --cached index.js

Commiting files from staging

git commit -m 'initial repo'

git log

Git Log online

git log --oneline
